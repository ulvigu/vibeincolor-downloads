Product information

Privacy.

Vibe in Color processes theme choices, font selections, and preferences locally on your Mac.

Last updated · July 22, 2026

App privacy

The app contains no analytics, telemetry, advertising, accounts, or tracking. It makes no network requests.

Vibe in Color uses information on your Mac only to provide its features:

  • It reads the theme library bundled inside the app.
  • It enumerates installed font names so you can preview and select a font face.
  • It stores font selections, favorites, and preview preferences in local app storage.
  • It writes a generated codex-theme-v1 value to the macOS clipboard when you choose Copy for ChatGPT.

Data transmission

The app does not transmit theme data, font information, preferences, favorites, or clipboard contents anywhere. Removing the app and its local preferences removes the information it stores.

Website and downloads

This product page is hosted by GitHub Pages, and installers are delivered through GitHub Releases. GitHub may process standard web request information when you visit or download a file under its own privacy statement. Vibe in Color does not add analytics to the website.
