Useful details

Including these details makes a report much easier to act on:

  • Vibe in Color version and macOS version.
  • ChatGPT for macOS version and build when the issue involves importing a theme.
  • Clear steps that reproduce the problem.
  • A screenshot when it helps explain what appeared on screen.

Quick import check

  1. Choose a theme in Vibe in Color and select Copy for ChatGPT.
  2. Open ChatGPT for macOS → Settings → Appearance.
  3. Use Import on the matching Light theme or Dark theme card, paste, and choose Import theme.

Version 0.1.1 was tested with ChatGPT for macOS 26.715.72359, build 5718.
